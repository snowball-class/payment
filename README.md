# 프로젝트 소개
## 프로젝트 개요
유튜브에 올라와 있는 영상들을 기반으로 온라인 강의를 들을 수 있는 프로젝트를 진행
해당 프로젝트에서 결제 서비스를 담당하는 부분입니다.
